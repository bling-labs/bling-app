# 인플루언서 목록·상세 페이지 구상 (추천)

콘텐츠 라이센싱 마켓플레이스 성격에 맞춘 인플루언서 목록·상세 화면 추천안입니다.

---

## 1. 헤더 동작 (적용 완료)

- **콘텐츠** → `/contents` (콘텐츠 목록 페이지)
- **컨테스트** → `/contest` (컨테스트 목록 페이지)
- **인플루언서** → `/influencers` (인플루언서 목록 페이지)
- 탭 클릭 시 해당 목록 페이지로 이동하며, 앵커 스크롤은 사용하지 않음.

---

## 2. 인플루언서 목록 페이지 (`/influencers`)

**역할:** 광고주가 크리에이터를 탐색·비교하고, 프로필·콘텐츠를 빠르게 훑어볼 수 있는 진입점.

### 추천 구성

| 영역 | 내용 |
|------|------|
| **헤더** | 타이틀 "인플루언서", 서브카피 "검증된 크리에이터 프로필을 확인하고 콘텐츠를 둘러보세요" |
| **필터** | 카테고리(#태그), 팔로워 범위, 콘텐츠 수, 소셜 연동/콘텐츠 전용, 정렬(추천·인기·최신·콘텐츠 많은순·평점순). 스크롤 시 상단 고정(sticky). |
| **카드** | 대표 이미지/영상(호버 시 재생), 프로필 이미지, 이름, @핸들, 카테고리 칩, 팔로워 수, 활동 플랫폼 아이콘(최대 3개 + 초과 시 "+ N"). 클릭 시 **상세로 이동**. |
| **그리드** | 데스크톱 4열, 태블릿 2열, 모바일 1열. 무한 스크롤 또는 "더 보기". |
| **빈/에러** | 필터 결과 없음 시 안내 + "필터 초기화". |

이미 구현된 목록 페이지가 위 스펙과 크게 다르지 않다면, 필터·카드 디테일만 PRD/디자인에 맞춰 조정하면 됩니다.

## 3. 인플루언서 상세 페이지

### 추천: 인터셉팅 라우트 + 모달

- **라우트:** `/influencers/[id]`  
  - 목록에서 카드/CTA 클릭 시 → 같은 도메인 내에서 ` /influencers/[id]` 로 이동.
  - 이때 Next.js **인터셉팅 라우트** `@modal/(.)influencers/[id]/page.tsx` 로 **같은 페이지 위에 모달** 렌더링.
- **동작:**
  - 목록 페이지는 뒤에 그대로 두고, 위에 모달만 열림.
  - 모달 닫기(또는 백드롭 클릭) 시 `/influencers`로 복귀 → 목록 유지.
  - 새로고침 또는 직접 `/influencers/[id]` 접근 시에는 **전체 페이지 상세**로 렌더 (모달 없이 풀페이지).

이렇게 하면 "목록 유지 + 빠른 비교"와 "딥링크/공유"를 둘 다 만족시킬 수 있습니다.

### 모달(상세) 내부 구성 추천

| 블록 | 내용 |
|------|------|
| **상단** | 커버/대표 영상 또는 이미지, 우측 상단 닫기(X). (선택) 재생/일시정지. |
| **프로필** | 프로필 이미지, 이름, @핸들, 카테고리(칩), 팔로워 수, 활동 플랫폼 아이콘, 한 줄 소개(있을 경우). |
| **콘텐츠 갤러리** | "이 크리에이터의 콘텐츠" — 썸네일 그리드(3~4열). 클릭 시 **콘텐츠 상세(아래 4번)** 로 연결. |
| **CTA** | "콘텐츠 둘러보기" → 해당 크리에이터 콘텐츠만 필터한 `/contents?creator=id` 이동, 또는 "문의하기"(추후 연동). |

데스크톱은 모달 중앙, 모바일은 하단 시트 또는 풀스크린에 가깝게 올라오는 형태로 구현해도 됩니다.

## 4. 콘텐츠 상세 페이지

**역할:** 특정 숏폼/이미지 콘텐츠의 라이센스 조건 확인, 크리에이터 프로필로 이동, "라이센스 문의/구매" 등 전환.

### 추천

- **라우트:** `/contents/[id]` (전용 페이지 권장).
- **구성:**  
  - 재생/이미지 영역, 제목·설명, 카테고리·태그.  
  - 크리에이터 블록(프로필 이미지, 이름, @핸들) → 클릭 시 **인플루언서 상세(모달 또는 `/influencers/[id]`)** 로 이동.  
  - 가격/라이센스 옵션(독점·비독점, 기간 등), CTA "라이센스 문의" / "장바구니" 등.
- 인플루언서 상세를 모달로 쓰는 경우, 콘텐츠 상세에서 "크리에이터 프로필" 클릭 시 같은 모달 패턴(`/influencers/[id]`)을 열어 흐름을 통일할 수 있습니다.

---

## 5. 정리

| 화면 | 라우트 | 형태 | 비고 |
|------|--------|------|------|
| 인플루언서 목록 | `/influencers` | 전용 페이지 | 필터·카드·무한스크롤 (현행 유지·보완) |
| 인플루언서 상세 | `/influencers/[id]` | **모달**(목록에서 진입) + 풀페이지(직접 접근) | 인터셉팅 라우트로 모달 추천 |
| 콘텐츠 상세 | `/contents/[id]` | 전용 페이지 | 크리에이터 블록에서 인플루언서 상세(모달) 연결 |

헤더의 "인플루언서"는 이미 `/influencers`로 연결되어 목록 페이지가 열리도록 수정되어 있습니다. 인플루언서 상세는 위와 같이 모달로 구현하는 구성을 추천합니다.
