# Bling 프로젝트 개요

## 프로젝트 정보

- **제품명**: Bling ✨
- **제품 유형**: 콘텐츠 라이센싱 플랫폼 (풀 웹서비스)
- **타겟 사용자**: 광고주/마케터, 인플루언서/크리에이터

## 서비스 구조

이 프로젝트는 세부 기능이 동작하는 **온전한 웹서비스**이며, 현재 메인 화면은 랜딩페이지 성격의 홈 화면이다.

### 상단 탭 네비게이션

- **콘텐츠** — 숏폼 콘텐츠 조회·탐색
- **컨테스트** — 광고주가 등록한 공모전 목록 (숏폼 광고 콘텐츠 공모)
- **인플루언서** — 인플루언서 목록 조회

### 주요 화면

| 탭 | 페이지 | 설명 |
|----|--------|------|
| (홈) | 메인 화면 | 랜딩 스타일, 플랫폼 소개·가치 전달 |
| 콘텐츠 | `/contents` | 숏폼 콘텐츠 나열·필터·라이센스 구매 |
| 컨테스트 | `/contest` | 숏폼 광고 콘텐츠 공모전 목록 |
| 인플루언서 | `/influencers` | 인플루언서 프로필 목록 |

## 사용자 유형

- **광고주/마케터**: 콘텐츠 라이센스 구매, 공모전 등록
- **인플루언서/크리에이터**: 콘텐츠 업로드·판매, 공모전 참여  
  - 소셜 계정 연동 없이 **콘텐츠만 업로드하는 크리에이터**도 가능

## 핵심 가치

- 인플루언서·광고주 연결 마켓플레이스
- 숏폼 영상·이미지 콘텐츠 라이센싱
- 공모전을 통한 숏폼 광고 콘텐츠 수집
