# Bling 메인 화면 (홈)

랜딩페이지 성격의 메인 화면. 플랫폼 가치 전달 및 광고주·인플루언서 양측 회원가입 유도.

## 접근

- URL: `/` (루트)
- 상단 탭: 홈(로고 클릭) 또는 콘텐츠/컨테스트/인플루언서

## 네비게이션

- 로고: "Bling" ✨
- 탭: **콘텐츠**, **컨테스트**, **인플루언서**
- 우측: 로그인, 회원가입

## 필수 섹션 (위→아래)

1. **히어로** — 헤드라인, 가치 제안, 이중 CTA (광고주/인플루언서)
2. **플랫폼 통계** — 인플루언서 수, 콘텐츠 수, 거래 건수, 거래액
3. **이중 가치 제안 카드** — 광고주용·인플루언서용 카드
4. **인기 인플루언서 쇼케이스** — 프로필 카드 그리드
5. **인기 콘텐츠 쇼케이스** — 카테고리 필터 + 콘텐츠 카드
6. **광고주·인플루언서 프로세스** — 4단계 타임라인 (탭 전환)
7. **최종 CTA** — 프로모션 + 회원가입 유도
8. **푸터** — 링크, 저작권

## CTA

- "광고주로 시작하기", "인플루언서로 시작하기"
- "콘텐츠 둘러보기", "수익 시뮬레이터", "라이센스 구매" 등
