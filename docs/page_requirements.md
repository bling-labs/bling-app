## 우선순위 요약

### 🔴 1순위 - MVP 필수 (서비스 런칭 최소 요건)
1. 로그인/회원가입 (광고주·크리에이터 분리)
2. 콘텐츠 목록 페이지 (`/contents`) — PRD 정의 완료, 미구현
3. 광고주 마이페이지 (기본 프로필, 라이센스 관리)
4. 크리에이터 스튜디오 (프로필, 콘텐츠 업로드/관리, 수익)
5. 이용약관/개인정보처리방침

### 🟡 2순위 - 핵심 사용성
6. 장바구니/체크아웃
7. 메시지/문의 시스템
8. 공모전 등록(광고주)/지원(크리에이터)
9. 통합 검색
10. 온보딩 플로우
11. FAQ/고객센터/가격 정책

### 🟢 3순위 - 성장 단계
12. 관리자 페이지
13. 분석/리포트
14. 고급 기능 (AI 추천, 비교, 팀 협업 등)
