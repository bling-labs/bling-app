## 페이지 개요

**목적:**  
광고주가 인플루언서/크리에이터 프로필을 탐색하고 필터링하여 원하는 크리에이터를 찾아 콘텐츠를 둘러보거나 라이센스 협의를 진행할 수 있는 페이지

**접근 경로:**
- 상단 네비게이션 "인플루언서" 탭 클릭
- 메인 화면 인플루언서 쇼케이스 "전체 보기" 클릭
- URL: `/influencers`

**주요 사용자:**
- 광고주/마케터 (크리에이터 탐색, 콘텐츠 라이센스 구매)
- 인플루언서/크리에이터 (경쟁자·동료 프로필 확인)

**참고:** 인플루언서는 소셜 연동 크리에이터와 콘텐츠만 업로드하는 크리에이터 모두 포함.

## 페이지 구조

### 1. 네비게이션 바 (공통)
- 로고: "Bling" ✨
- 탭 메뉴: 콘텐츠, 컨테스트, **인플루언서** (활성)
- 우측: 로그인, 회원가입 (비로그인 시) / 프로필, 알림 (로그인 시)

### 2. 페이지 헤더
**구성 요소:**
- 페이지 타이틀: "인플루언서"
- 서브타이틀: "검증된 크리에이터들의 프로필을 확인하고 콘텐츠를 둘러보세요"
- 검색바 (옵션): 닉네임·핸들 키워드 검색

### 3. 필터 영역
**레이아웃:** 가로 스크롤 가능한 태그 칩 + 추가 필터 드롭다운

**카테고리 필터 (태그 기반):**
- [전체] [뷰티] [패션] [푸드] [피트니스] [여행] [테크] [라이프스타일] [게임] [음악]
- 기본 선택: "전체"
- 활성 상태: 그라데이션 배경 (#3B82F6 → #8B5CF6)
- 비활성 상태: 투명 배경 + 테두리
- 태그명 앞에 # 붙임

**추가 필터 옵션 (드롭다운):**
- 팔로워 수: 전체 / 1K 이하 / 10K 이하 / 100K 이하 / 1M 이하 / 1M 이상
  - 소셜 미연동 크리에이터는 "팔로워 없음"으로 분류 또는 필터 결과에서 제외 옵션
- 콘텐츠 수: 전체 / 10개 이상 / 50개 이상 / 100개 이상
- 정렬: 추천순 / 인기순 / 최신 가입순 / 콘텐츠 많은순 / 평점순
- 타입: 전체 / 소셜 연동 / 콘텐츠 전용 (소셜 미연동)

**UI 위치:**
- 페이지 상단, 그리드 바로 위
- 스크롤 시 상단 고정 (sticky)

### 4. 인플루언서 그리드 영역

**레이아웃:** 카드 그리드

**반응형:**
- 데스크톱 (1280px+): 4열
- 태블릿 (768-1279px): 2열
- 모바일 (767px 이하): 1열

**그리드 간격:** 24px

## 인플루언서 카드 상세 스펙

### 카드 구조 (가로형 프로필 카드)

```
┌─────────────────────────────────────┐
│ [커버 이미지 영역 h-32]              │
│   (선택) 또는 단색 그라데이션         │
└─────────────────────────────────────┘
│ [아바타 64px]  이름 / @핸들           │
│   -top-8      카테고리태그뱃지           │
│               ─────────────────      │
│               👥 팔로워           │
│               [프로필 보기] CTA       │
└─────────────────────────────────────┘
```

### 상세 요소 배치

**커버 이미지 (상단):**
- 높이: 128px (h-32)
- 비율: 16:9 또는 가로 전체
- 소스: 인플루언서가 설정한 커버 이미지, 없으면 그라데이션 (primary → secondary)
- 하단 그라데이션: from-card to-transparent

**아바타:**
- 위치: 커버 이미지 하단과 겹치게 (-top-8)
- 크기: 64x64px
- 스타일: 원형, 4px 카드 배경색 테두리

**이름:**
- 텍스트: 16px, 세미볼드, foreground
- 예시: "김수민"

**핸들:**
- 텍스트: 14px, primary 색상
- 예시: "@beautykim"

**카테고리 배지:**
- 스타일: 작은 칩, bg-muted, text-muted-foreground
- 예시: "뷰티·스킨케어", "피트니스·헬스"
- 복수 카테고리 시 대표 1개 또는 "뷰티, 패션" 형태

**통계 영역 (하단):**
- 팔로워: 👥 아이콘 + "85K" (소셜 연동 시) 또는 "—" (미연동 시)

**CTA 버튼:**
- 라벨: "프로필 보기" 또는 "콘텐츠 보기"
- 클릭 시: `/influencers/:id` 상세 페이지 이동

### 카드 호버 효과
- 테두리: border-primary/50
- 살짝 위로: translateY(-2px)
- 그림자 증가
- 전환: 0.2s ease

## 인터랙션 및 기능

### 카드 클릭 동작
- 클릭 시: 인플루언서 상세 페이지로 이동 (`/influencers/:id`)
- 상세 페이지: 프로필 정보, 콘텐츠 갤러리, 문의/협의 CTA

- 로그인 사용자만 인플루언서 즐겨찾기 가능
- 우상단 하트 아이콘

## API (향후 연동)

**GET `/api/influencers`**

쿼리 파라미터:
- `category`: 카테고리 (다중 가능)
- `followerRange`: 1k, 10k, 100k, 1m, 1m_plus
- `minContents`: 최소 콘텐츠 수
- `type`: all, social, content_only
- `sort`: recommended, popular, latest, contents_desc, rating
- `page`, `limit`

응답 예시:
```json
{
  "influencers": [
    {
      "id": "inf_1",
      "name": "김수민",
      "handle": "@beautykim",
      "profileImageUrl": "https://...",
      "coverImageUrl": "https://...",
      "category": "뷰티·스킨케어",
      "socialConnected": true,
      "followerCount": 85000,
      "contentCount": 124,
      "rating": 4.9
    }
  ],
  "totalCount": 89,
  "hasMore": true
}
```
