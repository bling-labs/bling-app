## 페이지 개요

**목적:**  
광고주가 구매 가능한 숏폼 영상 콘텐츠를 탐색하고 필터링하여 원하는 콘텐츠를 찾아 라이센스를 구매할 수 있는 페이지

**접근 경로:**
- 상단 네비게이션 "콘텐츠" 탭 클릭
- 메인 화면 "콘텐츠 둘러보기" CTA 클릭
- URL: `/contents`

**주요 사용자:**
- 광고주/마케터 (콘텐츠 구매자)
- 인플루언서/크리에이터 (자신의 콘텐츠 확인)

## 페이지 구조

### 1. 네비게이션 바 (공통)
- 로고: "Bling" ✨
- 탭 메뉴: **콘텐츠** (활성), 컨테스트, 인플루언서
- 우측: 로그인, 회원가입 (비로그인 시) / 프로필, 알림 (로그인 시)

### 2. 페이지 헤더
**구성 요소:**
- 페이지 타이틀: "콘텐츠" 또는 "숏폼 콘텐츠"
- 서브타이틀: "검증된 인플루언서의 숏폼 영상을 라이센스하세요"
- 검색바 (옵션): 키워드 검색 기능

### 3. 필터 영역
**레이아웃:** 가로 스크롤 가능한 태그 칩 리스트

**카테고리 필터 (태그 기반):**
- [전체] [뷰티] [패션] [푸드] [피트니스] [여행] [테크] [라이프스타일] [게임] [음악] [더보기...]
- 기본 선택: "전체"
- 활성 상태: 그라데이션 배경 (#3B82F6 → #8B5CF6)
- 비활성 상태: 투명 배경 + 테두리
- 다중 선택 가능 (OR 조건)
- 디자인도 태그명 앞에 #붙임

**추가 필터 옵션 (드롭다운 또는 사이드바):**
- 가격 범위 슬라이더
- 팔로워 수 범위(1k 이하, 10k이하, 100k이하 1m 이하, 1m이상)
- 정렬: 최신순 / 인기순 / 가격 낮은순 / 가격 높은순
- 라이센스 유형: 전체 / 독점 가능 / 비독점만

**UI 위치:**
- 페이지 상단, 콘텐츠 그리드 바로 위
- 고정 (스크롤 시 상단에 고정되도록)

### 4. 콘텐츠 그리드 영역

**레이아웃:** 3x3 그리드 (한 화면에 9개 콘텐츠)

**반응형:**
- 데스크톱 (1280px+): 3열
- 태블릿 (768-1279px): 2열
- 모바일 (767px 이하): 1열

**그리드 간격:** 24px

## 콘텐츠 카드 상세 스펙

### 카드 구조
각 콘텐츠는 **하나의 카드**로 표시되며, 카드는 **썸네일 이미지 위에 모든 정보가 오버레이**됩니다.

```
┌─────────────────────────────┐
│ [태그]           [가격범위]  │ ← 상단
│                             │
│      썸네일 이미지            │
│      (9:16 비율)            │
│                             │
│                             │
│ [▶]                         │ ← 중앙 재생 버튼
│                             │
│                             │
│ [@프로필] 닉네임              │ ← 하단
│ 팔로워 10K-50K (옵션)        │
└─────────────────────────────┘
```

### 상세 요소 배치

**좌상단 - 카테고리 태그:**
- 위치: 왼쪽 상단 모서리에서 12px 안쪽
- 스타일: 작은 칩 형태, 반투명 다크 배경 (#000000 60% opacity)
- 텍스트: 12px, 흰색, 볼드
- 예시: "뷰티", "푸드", "여행"
- 최대 1개 표시 (대표 태그)

**우상단 - 가격 범위:**
- 위치: 오른쪽 상단 모서리에서 12px 안쪽
- 스타일: 작은 칩 형태, 반투명 다크 배경 (#000000 60% opacity)
- 텍스트: 12px, 흰색, 볼드
- 아이콘: 💰 또는 ₩
- 표시 형식: "₩200K-500K" 
- 설명: 크리에이터가 설정한 라이센스 판매 가격 범위
  - 예: 독점 라이센스 ₩500K, 비독점 ₩200K → "₩200K-500K" 표시
  - 단일 가격만 있는 경우: "₩300K"

**중앙 - 재생 버튼:**
- 위치: 정중앙
- 스타일: 큰 원형 버튼 (64px 직경)
- 아이콘: ▶ (흰색)
- 배경: 반투명 흰색 (#FFFFFF 30% opacity)
- 호버 효과: 배경 불투명도 50%로 증가, 살짝 확대

**하단 - 크리에이터 정보:**
- 위치: 하단에서 12px 위, 왼쪽 정렬
- 배경: 그라데이션 오버레이 (투명 → #000000 80%)
- 구성:
  ```
  [프로필 이미지 32px] 닉네임
  팔로워 10K-50K (소셜 연동 시에만)
  ```
  
**프로필 이미지:**
- 크기: 32x32px
- 스타일: 원형, 2px 흰색 테두리

**닉네임:**
- 텍스트: 14px, 흰색, 세미볼드
- 예시: "@beautykim", "크리에이터A"

**팔로워 범위 (조건부 표시):**
- 조건: 인플루언서가 소셜 채널을 등록한 경우에만 표시
- 텍스트: 12px, 흰색 80% opacity
- 아이콘: 👥 또는 없음
- 표시 형식: "팔로워 10K-50K"
- 설명: 인플루언서의 등록된 소셜 채널 팔로워 수 범위
  - 예: 인스타그램 45K → "10K-50K" 범위로 표시
  - 소셜 미연동 크리에이터는 이 정보 없음

**오버레이:**
- 상단: 투명

## 인터랙션 및 기능

### 카드 호버 효과
- 카드 전체: 살짝 위로 올라가는 효과 (translateY -4px)
- 그림자: 더 깊어짐 (box-shadow 증가)
- 재생 버튼: 배경 불투명도 증가, 살짝 확대
- 전환: 0.2s ease

### 카드 클릭 동작
- 클릭 시: 콘텐츠 상세 페이지로 이동 (`/contents/:id`)
- 새 탭 열기: Ctrl/Cmd + 클릭

### 영상 미리보기 (고급 기능, 옵션)
- 호버 2초 후: 짧은 미리보기 영상 자동 재생 (음소거)
- 마우스 이탈 시: 원래 썸네일로 복귀

### 북마크/좋아요 (옵션)
- 위치: 우하단 또는 우상단
- 아이콘: ❤️ 또는 🔖
- 로그인 사용자만 사용 가능

## 데이터 표시 로직

### 가격 범위 계산
```javascript
// 크리에이터가 설정한 라이센스 옵션
licenses = [
  { type: "독점", price: 500000 },
  { type: "비독점_1개월", price: 200000 },
  { type: "비독점_6개월", price: 350000 }
]

// 표시: 최소-최대
display = "₩200K-500K"

// 단일 가격인 경우
licenses = [{ type: "비독점", price: 300000 }]
display = "₩300K"
```

### 팔로워 범위 표시 로직
```javascript
// 인플루언서가 소셜 채널 등록한 경우
if (creator.socialConnected) {
  followerCount = 45000
  
  // 범위로 표시 (프라이버시 보호)
  if (followerCount < 10000) display = "~10K"
  else if (followerCount < 50000) display = "10K-50K"
  else if (followerCount < 100000) display = "50K-100K"
  else if (followerCount < 500000) display = "100K-500K"
  else display = "500K+"
} else {
  // 소셜 미연동 크리에이터는 표시 안함
  display = null
}
```

### 태그 우선순위
- 콘텐츠에 여러 태그가 있을 경우, 대표 태그 1개만 표시
- 우선순위: 크리에이터가 지정한 주 카테고리 → 가장 많이 검색된 태그

## 기술 요구사항

### 프론트엔드
- React/Next.js
- 반응형 그리드: CSS Grid 또는 Flexbox
- 무한 스크롤: Intersection Observer API
- 이미지 최적화: Next.js Image 컴포넌트

### 백엔드 API
**GET `/api/contents`**

쿼리 파라미터:
- `category`: 카테고리 태그 (다중 가능, 쉼표 구분)
- `minPrice`, `maxPrice`: 가격 범위
- `minFollowers`, `maxFollowers`: 팔로워 범위
- `sort`: 정렬 (latest, popular, price_asc, price_desc)
- `page`: 페이지 번호
- `limit`: 한 페이지당 개수 (기본 9)

응답:
```json
{
  "contents": [
    {
      "id": "content_123",
      "thumbnailUrl": "https://...",
      "category": "뷰티",
      "priceRange": { "min": 200000, "max": 500000 },
      "creator": {
        "id": "creator_456",
        "nickname": "@beautykim",
        "profileImageUrl": "https://...",
        "socialConnected": true,
        "followerRange": { "min": 10000, "max": 50000 }
      },
      "tags": ["뷰티", "스킨케어", "루틴"]
    }
  ],
  "totalCount": 127,
  "hasMore": true
}
```
